This folder contains the curated seed layer for CoreInventory.

Files:
- `users.json`: demo identities and credentials used for seeding.
- `categories.json`: normalized product categories.
- `warehouses.json`: warehouse and location hierarchy.
- `products.json`: curated product master with stock-by-location allocations.
- `operations.json`: pending operational documents used to seed receipts, deliveries, transfers, and adjustments.
- `history.json`: curated stock movement ledger used by dashboard and move history screens.
- `source-manifest.json`: source URLs and sampling metadata for the current curated dataset.

The current data is generated by `scripts/curate_public_dataset.py`, which merges:
- Amazon Reviews 2023 `Industrial_and_Scientific`
- Amazon Reviews 2023 `Tools_and_Home_Improvement`
- UCI `Online Retail II`

The output is normalized to the app schema so it can be seeded directly into PostgreSQL and displayed in the main site.
